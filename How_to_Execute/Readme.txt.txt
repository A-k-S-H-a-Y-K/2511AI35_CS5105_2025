# How to Execute Student Grouping App

1. Clone the repository:

   git clone https://github.com/A-k-S-H-a-Y-K/2511AI35_CS5105_2025.git

2. Navigate to the folder:

   cd 2511AI35_CS5105_2025/How_to_Execute

3. Install dependencies:

   pip install -r requirements.txt

4. Run the Streamlit app:

   streamlit run distribution.py

5. Upload the input CSV when prompted and select the number of groups.

6. Press 'Process' to generate:

   - Uniform Distribution summary

   - Mix Distribution summary

7. Download the summaries using the 'Download CSV' buttons.
